## 练习1
* 创立数据库表 examstudent，表结构如下

    字段名 |说明 |类型
    :--- |:--- |:---
    FlowID |流水号 |int  
    Type |四级,六级 |int  
    IDCard |身份证 |varchar(18)  
    ExamCard |准考证 |varchar(15)  
    StudentName |姓名 |varchar(20)  
    Location |区域 |varchar(20)  
    Grade |成绩 |int
